A modern, responsive, and interactive portfolio website built to showcase my skills, projects, certifications, and professional journey as a Computer Science student and Full Stack Developer.

## 📌 About

This portfolio highlights my technical skills, academic background, projects, certifications, and contact information. It is designed with a clean UI, smooth animations, and a responsive layout to provide an excellent user experience across all devices.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Puspraj-Magar/my-portfolio.git
```

Navigate to the project

```bash
cd my-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
